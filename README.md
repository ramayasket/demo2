Проект демонстрирует программу для синтетического арбитража.

Программа построена как типовой микросервис на AspNet.Core.
Вместе с тем, в программе всего 100 строк исполняемого кода,
поэтому разбиение на несколько микросервисов не оправдано.

Программа выполняет вычисление раз в минуту, также можно вызвать вычисление через Web API (POST).
Другой метод (GET) позволяет получить последнее вычисленное значение.

Пользовательский интерфейс к Web API доступен при помощи Swagger.
